## khulnasoft-status

Shows status of the Khulnasoft local development stack.

Requires the local development stack to be started by running `khulnasoft start` or `khulnasoft db start`.

You can export the connection parameters for [initializing khulnasoft-js](https://khulnasoft.com/docs/reference/javascript/initializing) locally by specifying the `-o env` flag. Supported parameters include `JWT_SECRET`, `ANON_KEY`, and `SERVICE_ROLE_KEY`.
